Sometimes, I put out demos as well: /demos

  • dbtree - A tool to visualize database schemas right inside your terminal.
  • pg_nowhere - A PostgreSQL extension to disallow UPDATE and DELETE queries without a WHERE clause.
  • newsbot - Automated news digest from Hacker News and Parallel AI, delivered to Slack.
  • numero - Parse and run math expressions anywhere, available as a library and as a web service.
  • monkeyc - Compiler for the Monkey programming language, written in C, extends the idea to have a self-contained executable.
  • teachyourselfmath - A website to browse 1000s of math problems and create your own problem sets.
  • vn53 - A DNS resolver.
  • workdiff - Generate changelogs from commits and send them to Slack, on a daily/weekly/monthly basis.
  • trackbit - Monitor your network throughput in Windows and Linux.
  • retain - A key-value store that speaks RESP.
  • makesite - Static site generator.
  • hello - Personalized terminal dashboard.
  • eng-graphics - A command line utility to generate Engineering Drawing sketches based on user's requirements.
