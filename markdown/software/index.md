# software

Sometimes, I put out demos as well: [/demos](/demos/)

- [dbtree](https://vivekn.dev/dbtree/) - A tool to visualize database schemas right inside your terminal.
- [pg_nowhere](https://github.com/viveknathani/pg_nowhere) - A PostgreSQL extension to disallow UPDATE and DELETE queries without a WHERE clause.
- [newsbot](https://github.com/viveknathani/newsbot) - Automated news digest from Hacker News and Parallel AI, delivered to Slack.
- [numero](https://github.com/viveknathani/numero) - Parse and run math expressions anywhere, available as a library and as a web service.
- [monkeyc](https://github.com/viveknathani/monkeyc) - Compiler for the Monkey programming language, written in C, extends the idea to have a self-contained executable.
- [teachyourselfmath](https://github.com/viveknathani/teachyourselfmath) - A website to browse 1000s of math problems and create your own problem sets.
- [vn53](https://github.com/viveknathani/vn53) - A DNS resolver.
- [workdiff](https://github.com/viveknathani/workdiff) - Generate changelogs from commits and send them to Slack, on a daily/weekly/monthly basis.
- [trackbit](https://github.com/viveknathani/trackbit) - Monitor your network throughput in Windows and Linux.
- [retain](https://github.com/viveknathani/retain) - A key-value store that speaks RESP.
- [makesite](https://github.com/viveknathani/makesite) - Static site generator.
- [hello](https://github.com/viveknathani/hello) - Personalized terminal dashboard.
- [eng-graphics](https://github.com/viveknathani/eng-graphics) - A command line utility to generate Engineering Drawing sketches based on user's requirements.
